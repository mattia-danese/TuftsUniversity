API Information

The API I selected is bible-api.com. This API provides easy online access to all the Bible verses which can also be in different languages. Using this API will return, in JSON format, all the verses that was specified in the API call.

I found the API in this GitHub repo that is a collection of numerous public APIs.

The option I used for the API request was "john 3:16". This specifies the chapter and verse in that chapter that the API should return. This is the only option to specify in the API request.

Two applications where this API would be helpful are:
  • a site that when given a Bible verse translates it into a desired language

  • free digitization of the Bible so having a physical copy is not needed
